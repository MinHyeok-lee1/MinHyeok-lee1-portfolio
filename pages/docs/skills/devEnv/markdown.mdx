## 마크다운 문법

### 1. 헤더

- 1~6까지 지원

```
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

- 적용 예시
  # H1
  ## H2
  ### H3
  #### H4
  ##### H5
  ###### H6

### 2. 목록

#### 2-1. 순서 있는 목록

숫자와 점 사용(숫자는 1로만 입력해도 내림차순으로 정의)

```
1. 첫번째
    1. 첫번째
        1. 첫번째
        1. 두번째
    1. 두번째
        1. 세번째
        1. 네번째
2. 두번째
3. 세번째
```

- 적용 예시

1. 첫번째
   1. 첫번째
      1. 첫번째
      1. 두번째
   1. 두번째
      1. 세번째
      1. 네번째
2. 두번째
3. 세번째

#### 2-2. 순서 없는 목록

- (글머리 기호 \*, +, - 지원)

```
* 1
    * 2
        * 3

- 1
    - 2
        - 3

혼합해서 사용 가능함

* 1
    - 2
        + 3
            * 4

```

- 적용 예시

* 1
  - 2
    - 3

- 1
  - 2
    - 3

* 1
  - 2
    - 3
      - 4

### 5. 이미지

```
![title](이미지 경로)
ex) ![토바기](/image.png)

* 사이즈 조절
<img src="이미지경로" width="oo px" height="oo px" title="oo"></img>
<img src="이미지 경로" width="oo %" height="oo %" title="oo"></img>

ex) <img src="/image.png" width="90px" height="90px" title="image"></img>
    <img src="/image.png" width="50%" height="50%" title="image"></img>
```

- 적용 예시

|     일반 이미지 삽입      |                           픽셀 사이즈 조정                            |                            % 사이즈 조정                            |
| :-----------------------: | :-------------------------------------------------------------------: | :-----------------------------------------------------------------: |
| ![image](/image.png)<br/> | <img src="/image.png" width="90px" height="90px" title="image"></img> | <img src="/image.png" width="50%" height="50%" title="image"></img> |

## pnpm 활용

### 의존성 설치

`pnpm install` 또는 `pnpm i` 명령어는 프로젝트의 `package.json` 파일에 정의된 모든 의존성을 설치한다.

```bash
pnpm i
```

이 명령어를 사용하면 필요한 모든 패키지가 설치되며, 프로젝트를 빌드하거나 개발 서버를 실행할 수 있게 된다.

### 개발 서버 시작

`pnpm dev` 명령어는 개발 서버를 시작하여 로컬 환경에서 MDX 파일의 변경 사항을 실시간으로 확인할 수 있게 해준다.

```bash
pnpm dev
```

이 명령어는 프로젝트에서 설정된 개발 서버를 실행하여, 브라우저에서 프로젝트를 열어 실시간으로 문서 또는 애플리케이션의 변경 사항을 확인할 수 있게 한다.

### 요약

- **Node.js**: JavaScript 런타임 환경으로, 서버 측 애플리케이션을 개발할 수 있게 해준다.
- **pnpm**: 빠르고 효율적인 패키지 매니저로, 의존성 설치 및 관리에 사용된다.
- **pnpm install**: 프로젝트의 모든 의존성을 설치한다.
- **pnpm dev**: 개발 서버를 시작하여 실시간으로 MDX 파일의 변경 사항을 확인할 수 있다.

이 과정들을 통해 MDX 파일을 효과적으로 작성하고, 로컬에서 확인할 수 있다.
