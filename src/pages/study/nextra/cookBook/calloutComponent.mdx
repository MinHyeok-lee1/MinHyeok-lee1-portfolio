# Callout 컴포넌트

Callout 컴포넌트는 독자에게 중요한 정보를 보여주는 기본 제공 구성 요소이다.

## Callout 컴포넌트 사용법

- 문서 상단에 Callout 컴포넌트를 import 한다.

```js
import { Callout } from "nextra-theme-docs";
```

### Default Callout Component

예제:

```js
<Callout type="default">default 타입의 콜아웃 컴포넌트의 모양이다.</Callout>
```

결과:

<Callout type="default">default 타입의 콜아웃 컴포넌트의 모양이다.</Callout>
