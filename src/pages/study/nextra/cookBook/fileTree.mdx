import { FileTree } from "nextra-theme-docs";

# FileTree

FileTree는 폴더와 파일의 구조를 시각적으로 표현한다.

## FileTree 사용법

- 문서 상단에 FileTree 컴포넌트를 import 한다.

```js
import { FileTree } from "nextra-theme-docs";
```

예제:

```
<FileTree>
  <FileTree.Folder name="pages" defaultOpen>
    <FileTree.File name="_meta.json" />
    <FileTree.File name="about.mdx" />
    <FileTree.File name="contact.mdx" />
    <FileTree.File name="index.mdx" />
  </FileTree.Folder>
</FileTree>
```

결과:

<FileTree>
  <FileTree.Folder name="pages" defaultOpen>
    <FileTree.File name="_meta.json" />
    <FileTree.File name="about.mdx" />
    <FileTree.File name="contact.mdx" />
    <FileTree.File name="index.mdx" />
  </FileTree.Folder>
</FileTree>

- defaultOpen 옵션은 기본적으로 폴더가 열려 내부 구성을 보여줄지 보여주지 않을지 선택하는 옵션이다.
- defaultOpen 옵션을 붙이지 않으면 위 코드는 초기에 아래와 같이 출력된다.
- 폴더를 클릭하면 내부 구성이 열린다.

<FileTree>
  <FileTree.Folder name="pages">
    <FileTree.File name="_meta.json" />
    <FileTree.File name="about.mdx" />
    <FileTree.File name="contact.mdx" />
    <FileTree.File name="index.mdx" />
  </FileTree.Folder>
</FileTree>
