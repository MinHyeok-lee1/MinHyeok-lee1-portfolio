# Flutter 개발환경 구축 및 설정

## Flutter 개발환경 구축

### 1. Flutter 프로젝트 설치
- [Flutter 설치 페이지](https://flutter-ko.dev/get-started/install)에서 본인의 os에 해당하는 운영체제를 선택한다.
- Flutter zip을 다운받고 압축해제한다.
- 검색 시작 표시줄에서 'env'를 입력하고 계정의 환경 변수 편집을 선택하고 flutter\bin이 위치한 파일경로를 저장한다.

<br />

### 2. 설치가 잘 되었는 지 아래 명령어로 확인한다.

```sh
$ flutter doctor
```
- 설치가 잘 되었다면 flutter 관련 진단 명령창이 출력된다.
- 체크가 되지 않고 X표시가 된 목록들을 설치하거나 설정을 해주어 해결해야 한다.

<br />

### 3. Android Studio 설치

- [Android Studio 설치 페이지](https://developer.android.com/studio)에서 Android Studio를 설치한다.

<br />

### 4. cmdline-tools component is missing flutter 에러 해결방법

- AndroidStudio를 실행한다.
- More Actions을 선택한다.
- SDK Manager를 클릭한다.
- SDK Tools 클릭 후 Android SDK Command-line Tools를 설치한다.

<br />

### 5. Android license status unknown 해결방법
```sh
$ flutter doctor --android-licenses
```
- 위 명령어 실행 후 y를 계속 눌러 licenses에 동의하면 해결된다.

<br />

### 8. 위 3~5의 과정이 잘 되었는 지 아래 명령어로 확인한다.

```sh
$ flutter doctor
```
- 체크가 되지 않고 X표시가 된 목록들을 설치하거나 설정을 해주어 해결해야 한다.
- Visual Studio를 제외한 모든 항목이 정삭적으로 체크되어 있어야 한다.

<br />

### 9. Flutter 플러그인 설치
- 안드로이드 스튜디오 플러그인에서 Flutter 검색 후 설치한다.

### 10. Flutter 프로젝트 실행
- 안드로이드 스튜디오에서 Flutter Project를 실행할 수 있다.
