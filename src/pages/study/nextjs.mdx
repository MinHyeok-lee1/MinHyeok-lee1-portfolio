# Next.js 문서구조

## Next.js란
Next.js는 Node.js 위에서 빌드된 오픈 소스 웹 개발 프레임워크이다.  
서버 사이트 렌더링, 정적 웹 페이지 생성 등 리액트 기반 웹 애플리케이션 기능을 제공한다.  


## 요구사항

- JavaScript
- React.js
