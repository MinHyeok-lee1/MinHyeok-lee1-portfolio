# 이미지 사용법

Next.js에는 이미지를 사용하는 방법은 `<Image>`태그를 사용하는 것이다.

## 1. Next.js `<Image>` 태그

기본적인 Next.js에서 `<Image>` 태그 사용방법.

예제:

```js
<Image src="/example.jpeg" alt="ERROR" width={500} height={500} />
```

결과:

<Image src="/example.jpeg" alt="ERROR" width={500} height={500} />

- 위와 같이 Next.js의 Image를 사용하면 레이아웃 이동이 없다.

- 크기(width와 height)를 정해줘야 한다.
