# NestJS 문서구조

## NestJS란

NestJS는 자바스크립트나 타입스크립트로 서버 애플리케이션을 개발할 수 있는 백앤드 웹 프레임워크이다.  
코드를 Provider, Controller, Module로 분리하며 컴포넌트들끼리의 조합(DI)을 통해 전체 애플리케이션을 완성한다.



## 요구사항

- JavaScript
- React.js
