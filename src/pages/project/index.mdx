# 개발 문서정리

## 개발문서 문서정리 내용

- 개발 문서정리의 컨텐츠는 지금까지 했던 프로젝트들의 설명들이 기술되어 있다.
- GitHub Repository의 권한 Public으로 설정되어있는 경우 설명과 프로젝트 링크가 첨부된다.
- GitHub Repository의 권한 Private로 설정되어있는 경우 프로젝트의 대략적인 폴더구조와 설명이 기술되어 있다.
- 혼자 100% 개발한 프로젝트가 아닌경우 맡은 부분과 사용한 기술들이 기술되어 있다.
