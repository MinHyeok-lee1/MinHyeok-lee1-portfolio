# Duobattlegame

- Flutter와 Dart를 처음으로 사용하여 앱 개발
- 하나의 휴대폰으로 두 플레이어가 대결하는 게임
  - Game1: 양쪽을 붙잡고 카운트가 끝나면 빠르게 누르는 사람이 이김
  - Game3: 양쪽을 붙잡고 하나의 길을 제한 시간안에 더 많이 누른사람이 이김
  - Game2: 양쪽을 붙잡고 5개의 길을 빠르게 눌러 한 쪽 길을 빠르게 미는 사람이 이김
  - Game4: 양쪽을 붙잡고 5개의 길을 누르면 상대방을 공격, 상대방 공격이 있다면 수비로 하여 공격을 받으면 짐
- Duobattlegame-APP 전체 담당
