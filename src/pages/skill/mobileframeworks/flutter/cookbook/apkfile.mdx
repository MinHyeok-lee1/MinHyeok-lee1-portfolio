# apk파일 추출하기

## 앱 빌드버전 다른 사람한테 공유할 때 apk 파일 추출하는 방법

- 안드로이드 스튜디오에서 플러터를 사용하여 개발하고 출시전 apk파일로 다른 사람한테 공유할 때 apk 파일을 추출하는 방법은 다음과 같다.

1. 안드로이드 스튜디오 상단바에 Build를 클릭한다.
   ![android header](/androidheader.png)
2. Flutter 선택 후 Build APK 파일을 선택하면 된다.
   ![android header](/androidapkfile.png)
3. APK 파일의 다운로드 위치: StudioProjects\flutter_app\build\app\outputs\flutter-apk
4. 이를 압축하여 다른 사람한테 공유하면 된다.
