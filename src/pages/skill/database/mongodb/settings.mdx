### 1. MongoDB 프로젝트 설치

Nest CLI를 사용하여 시작 프로젝트를 생성한다.  
프로젝트를 설치하고자 하는 디렉토리에서 아래 명령어를 MongoDB 프로젝트 폴더를 설치한다.

```sh
$ npm i -g @MongoDB/cli
$ nest new project-name
```

### 2. MongoDB Compass

### MongoDB 사용이유 및 설치

MongoDB Compass란?  
MongoDB Compass는 MongoDB 데이터를 쿼리, 최적화 및 분석하기 위한 무료 대화형 도구이다.  
핵심 인사이트를 얻고, 끌어서 놓기로 파이프라인을 구축하는 등의 작업을 수행할 수 있다.

MongoDB에 의해 구축된 GUI인 Compass에서 데이터로 쉽게 작업할 수 있다.  
Compass는 스키마 분석에서 인덱스 최적화, 집계 파이프라인에 이르기까지 모든 것을 중앙 집중식 단일 인터페이스에서 제공한다.

- Compass는 무료로 다운로드하여 사용할 수 있다.
- Linux, Mac 및 Windows에서 사용 가능하다.
- [해당 홈페이지](https://www.mongodb.com/products/compass)에서 MongoDB Compass를 설치할 수 있다.

#### MongoDB Compass 기능

1. 데이터 내에서 패턴 및 이상값 발견
   - 문서 스키마를 분석하여 데이터를 더 심층적으로 이해할 수 있다.
   - Compass는 문서를 샘플링하고 분석하여 날짜 및 정수 범위, 가장 빈번한 값 등을 포함하여 컬렉션에 대한 자세한 메타데이터를 제공한다.
2. 손쉽게 데이터 쿼리
   - Compass에 내장된 쿼리 표시줄을 사용하여 몇 초 만에 필요한 데이터를 찾을 수 있다.
   - 다른 언어의 표현식에 해당하는 직관적인 쿼리 연산자를 사용하여 컬렉션 내의 문서를 필터링한다.
   - 매우 세밀하게 결과를 샘플링, 정렬 및 수정한다.
3. 데이터베이스를 원활하게 실행하도록 유지
   - 새 인덱스를 추가하고 성능이 낮은 인덱스를 제거하여 쿼리를 최적화한다.
   - 실시간 서버 및 데이터베이스 메트릭을 모니터링한다.
   - 시각적 설명 계획으로 성능 문제를 조사하는 등의 작업을 수행한다.
4. 데이터를 집계하는 간단한 방법
   - 포함된 직관적인 빌더를 사용하여 몇 번의 클릭만으로 강력한 집계 파이프라인을 구성한다.
   - 파이프라인에 순차적 단계를 추가하고 각 단계를 진행하면서 데이터를 자동으로 미리본다.
   - 연간 지역 수익, 사용자 이탈 등과 같은 핵심 인사이트를 추출한다.
