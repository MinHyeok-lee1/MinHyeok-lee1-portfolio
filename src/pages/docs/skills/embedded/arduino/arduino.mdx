# Arduino 문서 시작

## 아두이노란?

\*\*아두이노(Arduino)\*\*란, **초보자부터 전문가까지 누구나 쉽게 전자 회로와 프로그래밍을 통해 하드웨어를 제어할 수 있도록 설계된 오픈소스 하드웨어 플랫폼**

---

### 🧩 1. 아두이노의 정의

> **Arduino**는 마이크로컨트롤러(MCU) 기반의 보드와 이를 쉽게 제어할 수 있는 개발 환경(IDE), 프로그래밍 언어, 다양한 라이브러리로 구성된 오픈소스 플랫폼

---

### 🛠️ 2. 구성 요소

| 구성           | 설명                                                     |
| -------------- | -------------------------------------------------------- |
| **하드웨어**   | Arduino Uno, Mega, Nano, Leonardo, ESP32 등 보드         |
| **소프트웨어** | Arduino IDE 또는 PlatformIO                              |
| **라이브러리** | 센서, 액추에이터, 통신 모듈 등을 쉽게 제어하는 코드 모음 |
| **언어**       | C/C++ 기반의 Arduino Language (`setup()`, `loop()` 구조) |

---

### 🔌 3. 무엇을 할 수 있을까?

- LED 점등부터 시작해…
- 온습도, 거리, 조도 등 센서 활용
- 모터, 서보, 릴레이 제어
- 블루투스/Wi-Fi를 이용한 무선 통신
- 자율주행 로봇, 스마트팜, IoT 기기 개발

---

### 🔍 4. 주요 특징

| 특징         | 설명                                           |
| ------------ | ---------------------------------------------- |
| 🧠 쉬운 입문 | 복잡한 설정 없이 바로 시작 가능                |
| 🌎 오픈소스  | 회로도, 소스코드 모두 무료 공개                |
| 📚 커뮤니티  | 수많은 예제와 문제 해결 자료 존재              |
| 🔌 확장성    | 센서/모듈과 쉽게 결합 (I2C, SPI, UART 등 지원) |

---

### ⚙️ 5. 아두이노의 작동 원리 (간단 흐름)

1. **회로 구성**
   → 센서, LED, 모터 등 아두이노 보드에 연결

2. **코드 작성**
   → `setup()`에서 초기 설정, `loop()`에서 반복 실행 코드 작성

3. **업로드**
   → USB로 코드 업로드

4. **실행 및 디버깅**
   → 시리얼 모니터 확인 또는 LED로 상태 체크

---

### 💡 6. 아두이노의 활용 분야

| 분야 | 예시                                  |
| ---- | ------------------------------------- |
| 교육 | 피지컬 컴퓨팅 수업, 입문용 코딩 교육  |
| 예술 | 미디어 아트, 인터랙티브 설치          |
| 산업 | 프로토타입 제작, 자동화 장치 개발     |
| DIY  | 스마트홈, RC카, 로봇 등 개인 프로젝트 |

---

### 🧠 7. 아두이노 vs 일반 MCU

| 비교 항목      | Arduino                | 일반 MCU (STM32 등)         |
| -------------- | ---------------------- | --------------------------- |
| 입문 난이도    | 매우 쉬움              | 중간\~어려움                |
| 개발 환경      | 통합 IDE (Arduino IDE) | 보통 복잡한 설정 필요       |
| 코드 복잡도    | 낮음 (초보자 친화적)   | 전반적으로 복잡함           |
| 성능/정밀 제어 | 낮음                   | 높음 (고성능 프로젝트 적합) |

---

### 📎 8. 대표 보드 비교

| 보드         | 특징                             |
| ------------ | -------------------------------- |
| Arduino Uno  | 가장 기본, ATmega328P 기반       |
| Arduino Mega | I/O 핀 많음, ATmega2560          |
| Arduino Nano | 소형화, Breadboard 호환          |
| ESP32        | Wi-Fi/Bluetooth 내장, 고성능 MCU |
