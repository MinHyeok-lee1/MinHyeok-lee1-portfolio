# 개발환경 설치 메뉴얼

[깃허브](https://github.com/)에서 repository를 pull한 이후 node.js기반 front-end 혹은 back-end를 사용하는 방법이다.

## Node.js Install

설치는 하단 링크에서 가능하다.  
[Node.js Install Link](https://nodejs.org/en/download/package-manager)

**설치 확인**: 터미널(또는 명령 프롬프트)을 열고 다음 명령어를 입력하여 설치가 제대로 되었는지 확인한다.

```bash
node -v
```

이 명령어를 실행하면 설치된 Node.js의 버전이 출력된다.

## pnpm Install

pnpm은 빠르고 효율적인 패키지 매니저로, Node.js 프로젝트에서 사용된다.  
`pnpm`을 사용하면 패키지 설치 속도가 빠르고, 디스크 공간을 절약할 수 있다.

- pnpm 설치 명령어  
  pnpm을 설치하기 위해 다음 명령어를 터미널에 입력한다.

```bash
npm install -g pnpm
```

이 명령어는 전역(global)으로 pnpm을 설치한다. 설치가 완료되면 `pnpm` 명령어를 사용할 수 있게 된다.
