# 구문 강조

Nextra는 Shiki를 사용하여 빌드 시 구문 강조를 수행한다.  
따라서 매우 안정적이고 성능이 좋다.  
아래와 같이 Markdown을 사용하여 구문을 추가할 수 있다.

```js
console.log("hello, world");
```

## 특징

### 인라인 코드

아래와 같이 인라인 구문 강조 표시는 ` `` ` 사이에 구문을 넣어 표시된다.  
Inlined syntax highlighting is also supported `let x = 1{:jsx}` via:

<br />

### 라인 강조

아래와 같이 `{}` 사이에 특정 줄 혹은 -를 사용하여 줄의 범위를 강조 표시할 수 있다.

```js {1,4-5}
import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);
  return <button onClick={() => setCount(count + 1)}>{count}</button>;
}
```

<br />

### 하위 문자열 강조 표시

아래와 같이 `//` 사이에 특정 문자열을 넣어 강조 표시를 할 수 있다.

```js /useState/
import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);
  return <button onClick={() => setCount(count + 1)}>{count}</button>;
}
```

<br />

### 복사 버튼 추가

아래 코드는 전역적으로 설정된 본 프로젝트에 `copy=false` 속성을 적용시킨 결과이다.

```js copy=false
console.log("hello, world");
```

Nextra 구성(파일)에서 설정하여 `defaultShowCopyCode: true` 기능을 전역적으로 위와 같이 활성화할 수 있다.
next.config.js에서 전역적으로 활성화되면 `copy=false` 속성을 통해 비활성화할 수 있다.

<br />

### 줄 번호 추가

아래 코드처럼 `showLineNumbers` 속성을 추가하여 코드 블록에 줄을 추가할 수 있다.

```js showLineNumbers
import { useState } from "react";

function Counter() {
  const [count, setCount] = useState(0);
  return <button onClick={() => setCount(count + 1)}>{count}</button>;
}
```

<br />

### 파일 이름 및 제목

아래 코드처럼 `filename` 속성 을 추가하여 코드 블록에 파일 이름이나 제목을 추가할 수 있다.

```js filename="example.js"
console.log("hello, world");
```
