# Link 사용법

모든 Markdown관련 링크는 자동으로 Next.js 링크로 변환되고 대상 페이지가 prefetch됨을 의미한다.  
링크를 클릭하면 전체 페이지를 로드하지 않고 SPA처럼 클라이언트 측에 페이지가 로드된다.

Nextra에는 이러한 링크를 구현하는 두 가지 방법이 있다.  
두 방법을 이용하여 루트 아래의 링크인 홈 (/)으로 링크 시켜주는 코드를 생성해보자.

## 1. Markdown 문법으로 Link 구현

예제:

```md
Click [here](/) to go home.
```

결과:

Click [here](/) to go home.

<br />

## 2. Next.js의 Link 컴포넌트로 Link 구현

예제:

```js
import Link from 'next/link'

Click <Link href="/">here</Link> to go home.
```

결과:

Click <Link href="/">here</Link> to go home.

<br />

- 위 출력결과에서 보면 알다싶이 here에 하이퍼링크가 표시되지 않는다. (here을 누르면 동작은 된다)
- Nextra에서는 Link 컴포넌트를 사용하기보다 Markdown 문법으로 링크를 걸도록 하자.
- 이 기능을 사용하면 Nextra 페이지 간을 빠르고 원활하게 탐색할 수 있다.
