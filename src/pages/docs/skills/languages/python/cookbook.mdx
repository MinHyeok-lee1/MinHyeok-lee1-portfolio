## 📌 16. 데코레이터 (Decorator)

```python
def decorator(func):
    def wrapper():
        print("전처리")
        func()
        print("후처리")
    return wrapper

@decorator
def say_hello():
    print("Hello!")

say_hello()
```

## 📌 17. 제너레이터 (Generator)

```python
def my_generator():
    yield 1
    yield 2
    yield 3

for val in my_generator():
    print(val)
```

- `yield`를 사용하면 값을 하나씩 반환하며 상태를 유지
- 메모리 효율적

## 📌 18. 이터레이터 (Iterator)

```python
nums = [1, 2, 3]
it = iter(nums)
print(next(it))  # 1
print(next(it))  # 2
```

- `__iter__()`와 `__next__()`를 가진 객체가 이터레이터

## 📌 19. 컨텍스트 매니저 (`with` 문)

```python
with open("file.txt", "r") as f:
    content = f.read()
```

사용자 정의 컨텍스트 매니저:

```python
class MyContext:
    def __enter__(self):
        print("시작")
        return self
    def __exit__(self, exc_type, exc_val, exc_tb):
        print("종료")

with MyContext():
    print("작업 중")
```

## 📌 22. 언패킹 연산자 `*`, `**`

## 📌 23. 조건 매칭 (match-case, Python 3.10+)

```python
def http_error(status):
    match status:
        case 400:
            return "Bad Request"
        case 404:
            return "Not Found"
        case 500:
            return "Server Error"
```
