# ❗ Dart 이슈 정리

## 1. Null check operator used on a null value
- `!` 연산자를 null인 값에 사용하면 런타임 오류 발생
```dart
String? name;
print(name!); // ❌ 오류
```
- ✅ 해결: null 확인 후 접근
```dart
if (name != null) {
  print(name);
}
```

---

## 2. LateInitializationError
- `late`로 선언하고 초기화 없이 사용 시 오류
```dart
late String name;
print(name); // ❌ 오류
```
- ✅ 생성자 또는 함수에서 초기화

---

## 3. NoSuchMethodError
- 정의되지 않은 메서드 호출 시 발생
```dart
var s = "hello";
s.add(1); // ❌
```

---

## 4. RangeError
- 리스트 인덱스 초과
```dart
var list = [1, 2, 3];
print(list[5]); // ❌ 오류
```

---

## 5. TypeError / CastError
- 타입 캐스팅이 잘못되었을 때
```dart
dynamic x = 'hello';
int y = x as int; // ❌
```

## 6. StateError
