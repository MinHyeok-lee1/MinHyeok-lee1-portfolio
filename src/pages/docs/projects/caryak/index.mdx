# CARYAK

- [CARYAK 웹사이트 ←](https://caryak.net/)
- (주) MK에서 개발한 자동차 정비 예약(고객 및 정비소) 웹사이트
- front-end, back-end, DevOps를 전반적으로 모두 다룸
  - front-end | VSC(VisualStudioCode)와 Next.js를 사용하여 개발
  - front-end | 정비소 웹사이트 예약 및 설정 페이지 제작
  - front-end | 접속 ip를 검사하여 접속자 파악 및 qc 웹 사이트 배포 웹사이트로 반환
  - 지역을 선택하여 차량 종류 및 정비 관련 일정을 예약함
  - 카카오 및 네이버 소셜 로그인 사용
  - back-end | VSC(VisualStudioCode)와 NestJS를 사용하여 개발
  - back-end | 페이지네이션 skip() 대신 hasNext()와 next()를 사용하여 속도 향상
  - back-end | 위도, 좌표값인 Geospatial Queries를 사용하여 index로 설정
  - back-end | Transaction을 사용하여 자원 임계구역 설정
  - database | mongoDB를 사용하여 데이터를 송, 수신 및 저장
  - Admin | front-end 및 back-end api 제작
  - DevOps 관리 및 유지보수 담당

### servers

카약 플랫폼을 구성하는 서버들과 서버 개발 및 운영에 활용되는 툴들로 구성

#### front-end

NextJS로 개발된 UI 제공용 서버

#### back-end

NestJS로 개발된 api 제공용 서버

#### database

MongoDB로 구성된 데이터베이스

#### model-builder

back-end의 ./src/models에 구현된 데이터 모델을 빌드하여 front-end로 공유할 수 있도록 하는 빌더

#### proxy-server

nginx 서버의 설정파일로 로컬 서버에서 dev.caryak.net을 구축하기 위해 활용

### devops

구현한 서버를 클라우드에 배포하고 관리하기 위한 툴과 자료들로 구성

### docs

프로젝트와 관련된 모든 자료들로 구성
