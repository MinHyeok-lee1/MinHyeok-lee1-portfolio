# i-PiE

- [아이파이 소개 홈페이지 ←](https://www.therobotics.kr/bbs/content.php?co_id=ipie)
- (주) 아트와에서 현대와 협업한 배수로 탐사로봇
- Embedded 부분에서 영상 처리 및 back-end, database 부분을 담당
  - Embedded | OrangePI에서 Python 프로그램으로 동작할 수 있게 개발
  - Embedded | AWS KVS를 사용하여 front-end로 영상을 송신
  - Embedded | MQTT를 사용하여 back-end로 로봇의 상태 및 제어명령을 송신
  - back-end | VSC(VisualStudioCode)와 NestJS를 사용하여 개발
  - back-end | 회원가입, 로그인등 Authorization 관련 api 개발
  - back-end | MQTT로 수신된 로봇의 상태 및 제어명령등을 front-end에서 요청을 받을 시 송신
  - database | mongoDB를 사용하여 데이터를 송, 수신 및 저장

### Embedded

로봇 아이파이의 메인로직을 수행하는 부분

### servers

아이파이의 조작을 위한 front-end, back-end, database로 구성

#### front-end

Svelte 로 개발된 UI 제공용 서버

#### back-end

NestJS로 개발된 api 제공용 서버

#### database

MongoDB로 구성된 데이터베이스
