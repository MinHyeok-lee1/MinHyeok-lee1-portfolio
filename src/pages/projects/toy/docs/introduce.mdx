# 개발새발

- 개발관련 기록, 포트폴리오, 학습노트 등을 목적으로 한 웹사이트
- .mdx로 구성되어 있어 깃허브의 .md 파일 작성 능력등 향상 도모

## 구축 환경

- Nextra로 개발
- Next.js와 .md파일 작성능력 요구

## 구성

개발새발 프로젝트는 다음과 같이 1종의 모듈로 구성된다.

- Nextra: Nextra를 사용하여 Next.js와 .mdx 두 가지를 사용하여 웹사이트 제공

## 모듈별 메뉴얼

### Nextra

Nextra에 필요한 2가지 기술들은 아래와 같다.

- Next.js
  - 기본적인 Next.js에서 pages와 component를 구성하는 능력
  - 아주 기본적인 Nextra만 사용할 때는 사용하지 않아도 됌
- .mdx
  - Next.js에서 제공하는 이미지, 라우팅등도 있지만 Nextra에서 .mdx에 유리하게 따로 제공
  - .md 파일 작성에 대한 기본 지식이 있어야 함
