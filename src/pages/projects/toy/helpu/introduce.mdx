# HelpU

- 한남대학교는 과사 홈페이지, 장학금 홈페이지, 메인 홈페이지등 정보가 너무 나뉘어져 있었음
- 과별로 정보를 크롤링하여 본인에게 맞는 과의 정보를 보여주자는 취지에서 시작한 토이 프로젝트
- 앱 제작 데이터 부분 및 레이아웃 구성하는 팀원 멘토와 Phython과 Selenium을 사용하여 데이터 크롤링
- 정보의 링크를 거는 것 그때 당시 개인정보보호법등의 문제가 되지 않았음
- 학교 측 허락을 받지 못해 불발

## 구축 환경

- HelpU-App: Android studio에서 java를 사용하여 구성, pythonPart에서 제공하는 데이터를 받아오는 FireBase를 사용
- pythonPart: VSC(VisualStudioCode)에서 Python과 Selenium을 사용

## 구성

### HelpU-App

- 한남대학교에서 과를 선택하여 과와 관련된 정보를 볼 수 있는 앱

### pythonPart

- 크롤링된 데이터를 FireBase를 통해 HelpU-APP에 제공
