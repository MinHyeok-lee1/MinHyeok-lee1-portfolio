# HelpU

- 한남대학교에서 과를 선택하여 과와 관련된 정보를 볼 수 있는 앱
- 한남대학교는 과사 홈페이지, 장학금 홈페이지, 메인 홈페이지등 정보가 너무 나뉘어져 있었음
- 과별로 정보를 크롤링하여 본인에게 맞는 과의 정보를 보여줌
- pythonPart에서 selenium을 사용하여 데이터를 크롤링하고 HelpU-APP에서 데이터를 받아오는 부분을 담당
