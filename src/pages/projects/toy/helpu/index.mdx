# HelpU

- 한남대학교 학과 별 정보 모음 모바일 애플리케이션
  - app | 크롤링한 데이터를 HelpU-APP에서 데이터를 받아오는 부분을 담당
  - app | 받아온 데이터를 보여주는 ui 레이아웃 개발
  - program | pythonPart에서 Selenium을 사용하여 데이터를 크롤링
