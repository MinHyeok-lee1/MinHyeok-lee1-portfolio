# IM (Insulin Manager)

- 다양한 과(경제학과, 멀티미디어 학과, 기계공학과 등)의 학생들이 참여한 캡스톤 디자인 프로젝트

## 구축 환경

- IM-APP: Android studio에서 java를 사용하여 구성
- MAIN-MCU: Arduino IDE를 사용하여 구성

## 구성

### IM-APP

- 인슐린 투여 환자가 인슐린을 투여할 시간을 예약하는 앱

### MAIN-MCU

- 예약된 시간이 되면 LED 및 부저로 알림을 주는 케이스의 MCU 메인로직 부분
