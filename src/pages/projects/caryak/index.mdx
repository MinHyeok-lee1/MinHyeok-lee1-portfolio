# CARYAK

- [카약 바로가기](https://caryak.net/)
- (주) MK에서 개발한 차량 정비관련 예약 플랫폼
- 지역을 선택하여 차량 종류 및 정비 관련 일정을 예약함
- 카카오 및 네이버 소셜 로그인 사용
- Front-end 로그인 및 쿠키관리, 관리자 페이지 개발
- Back-end, database 개발 및 관리
- DevOps 관리 및 유지보수 담당

### servers

카약 플랫폼을 구성하는 서버들과 서버 개발 및 운영에 활용되는 툴들로 구성

#### front-end

NextJS로 개발된 UI 제공용 서버

#### back-end

NestJS로 개발된 api 제공용 서버

#### database

MongoDB로 구성된 데이터베이스

#### model-builder

back-end의 ./src/models에 구현된 데이터 모델을 빌드하여 front-end로 공유할 수 있도록 하는 빌더

#### proxy-server

nginx 서버의 설정파일로 로컬 서버에서 dev.caryak.net을 구축하기 위해 활용

### devops

구현한 서버를 클라우드에 배포하고 관리하기 위한 툴과 자료들로 구성

### docs

프로젝트와 관련된 모든 자료들로 구성
