## 🧳 새로운 개발환경에서 alias 복사 방법

1. 아래 명령으로 기존 alias 백업:

```bash
alias > ~/my_aliases.sh
```

2. 새로운 환경에서 복원:

```bash
source ~/my_aliases.sh
```

3. 또는 `~/.zshrc`에 다음처럼 추가:

```bash
# 내 alias 모음
source ~/my_aliases.sh
```

## 📘 팁

* `alias | grep g` 로 git 관련 alias만 필터링 가능
* `which gco` 등으로 실제 명령어 확인 가능
* 새로운 명령어 추가는 `~/.zshrc` 하단에 `alias abbr='command'` 형식으로 등록
