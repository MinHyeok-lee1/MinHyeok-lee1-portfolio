## VSC에서 MarkDown 파일 Preview 여는 4가지 방법

🔹 방법 1: 단축키 사용 (가장 빠름)

- 파일을 열고 아래 단축키 입력

```bash
Mac:     Shift + Cmd + V
Windows: Ctrl + Shift + V
```

- 현재 `.md`파일의 오른쪽에 미리보기 창이 뜨고, 마크다운 렌더링된 결과를 보여줌

🔹 방법 2: 오른쪽 상단 아이콘 혹은 우클릭 OpenPreview

- 아래 두 가지중에 하나를 선택해서 적용
  1. README.md 파일을 열고 우측 위에 미리보기 아이콘 클릭
  2. REAME.md 파일을 열고 파일 우클릭하고 OpenPrewview 탭 클릭

🔹 방법 3: Command Palette로 열기

1. CMD 창에 아래 단축키 입력

   ```bash
   Mac:     Shift + Cmd + P
   Windows: Ctrl + Shift + P
   ```

2. Markdown: Open Preview to the Side 검색해서 클릭

🔹 방법 4: 확장 프로그램 사용

1. Markdown All in One 확장프로그램 다운로드
2. CMD 창에 아래 단축키 입력

   ```bash
   Mac:     Shift + Cmd + P
   Windows: Ctrl + Shift + P
   ```

3. 사용자 설정(UI) 실행
4. Markdown Prewview 입력 후 Auto Show Preview To Side 클릭
5. Markdown 파일 실행하면 자동으로 측면에 Preview가 열림

## Extensions

### 배포

- Docker

### MCU 개발

- PlatformIO IDE

### ROS 개발

- ROS
- Remote-SSH (Window 환경일 경우)

### 편의성

- Clang-Format : C/C++ 코드의 자동 포매팅 도구
- Material Icon Theme : 파일과 폴더를 위한 Material 디자인 아이콘 세트
- Tabnine : AI 기반 코드 자동 완성 기능
- Auto Close Tag : HTML/XML 태그를 자동으로 닫아주는 기능
- Auto Rename Tag : HTML/XML 태그를 동시에 수정해주는 기능
- Color Highlight : 코드 내 색상 코드에 대한 색상 미리보기 제공
- Commit Message Editor : 커밋 메시지 작성 시 도움을 주는 기능
- Error Lens : 코드의 오류를 강조 표시하여 가시성을 높임
- ESLint : JavaScript 및 TypeScript 코드의 linting과 오류 검출
- GitLens : Git 저장소의 히스토리와 통계 시각화
- Highlight Matching Tag : HTML/XML에서 태그 쌍을 강조 표시
- Image Preview : 코드 내 이미지 파일을 미리보기할 수 있는 기능
- IntelliCode : AI 기반 코드 추천 및 인사이트 제공
- markdownlint : Markdown 파일의 문법 오류를 검사하고 수정
- MDX : MDX 파일을 지원하여 Markdown과 JSX를 함께 사용 가능
- Path Intellisense : 파일 경로 자동 완성과 제안 기능
- Prettier : 코드의 일관된 스타일을 유지하는 자동 포매터
