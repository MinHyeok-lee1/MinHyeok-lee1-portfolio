# Step 컴포넌트

Step 컴포넌트는 번호가 매겨진 목록을 단계를 시각적으로 표현하는 문서 테마의 기본 제공 구성 요소이다.

## Step 컴포넌트 사용법

- 문서 상단에 Step 컴포넌트를 import 한다.

```js
import { Steps } from "nextra-theme-docs";
```

예제:

```mdx
<Steps>

### Step1

이것은 Step1에 관련된 설명이다.

### Step2

이것은 Step2에 관련된 설명이다.

### Step3

이것은 Step3에 관련된 설명이다.

</Steps>
```

결과:

<Steps>

### Step1

이것은 Step1에 관련된 설명이다.

### Step2

이것은 Step2에 관련된 설명이다.

### Step3

</Steps>
