# Nextra 문서 시작

## Nextra란

Nextra는 Next.js를 확장한 프레임워크이다.  
Markdown 형식의 일종인 mdx를 작성하는 방식으로 손쉽게 웹사이트를 구축할 수 있어 프로젝트 문서화 도구로 활용 가능하다.

- mdx: JSX를 포함할 수 있는 Markdown 컨텐츠

## 요구사항

- JavaScript
- React.js
- Next.js
- MarkDown
