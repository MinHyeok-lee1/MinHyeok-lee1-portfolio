# Raspberry Pi Pico 문서 시작

라즈베리파이 피코(Raspberry Pi Pico)는 라즈베리파이 재단에서 만든 **초소형, 고성능, 저전력의 마이크로컨트롤러 보드**  
일반적인 라즈베리파이(리눅스 보드)와는 달리, **MCU(Microcontroller Unit) 기반의 임베디드 시스템 보드**

---

## 📘 1. Raspberry Pi Pico란?

> **RP2040**라는 자체 개발 칩을 사용한 **마이크로컨트롤러 보드**로, 아두이노처럼 **센서 제어, 모터 구동, 통신** 등에 특화되어 있으며, **C/C++ 또는 MicroPython**으로 프로그래밍

---

## 🧠 2. 주요 사양

| 항목          | 사양                                      |
| ------------- | ----------------------------------------- |
| MCU           | **RP2040** (Dual-core ARM Cortex-M0+)     |
| 클럭 속도     | 최대 133 MHz                              |
| SRAM          | 264 KB                                    |
| 플래시        | 2 MB (외장 QSPI 플래시)                   |
| I/O 핀        | 26개 GPIO (PWM, ADC, SPI, I2C, UART 포함) |
| 아날로그 입력 | 3채널 12비트 ADC                          |
| USB           | USB 1.1 (디바이스 및 호스트 가능)         |
| 크기          | 21 x 51 mm                                |
| 가격          | 약 \$4 (저렴함)                           |

---

## 💻 3. 개발 환경

| 언어          | 개발 방법                                                                   |
| ------------- | --------------------------------------------------------------------------- |
| C/C++         | [Pico SDK](https://github.com/raspberrypi/pico-sdk), VS Code, Makefile 기반 |
| MicroPython   | Thonny IDE 또는 REPL 콘솔에서 인터프리터 방식으로 개발                      |
| Arduino IDE   | 추가 보드 매니저 설치로 지원 가능 (공식은 아님)                             |
| CircuitPython | Adafruit의 파생 MicroPython 사용 가능                                       |

---

## 🔌 4. 핀맵 개요

- 26개의 GPIO 핀 (PWM, SPI, UART, I2C 다중 기능 제공)
- 3개의 아날로그 입력 (ADC0\~ADC2)
- 16개 PWM 채널
- PIO (Programmable I/O)로 SPI/I2C/UART 등을 직접 구현 가능

👉 [공식 핀맵 PDF](https://datasheets.raspberrypi.com/pico/Pico-R3-A4-Pinout.pdf) 참고

## ⚙️ 5. 사용 예시

| 예제           | 설명                                      |
| -------------- | ----------------------------------------- |
| LED 깜빡이기   | GPIO 출력, `gpio_put()`                   |
| 센서 읽기      | ADC 또는 I2C로 거리, 온도 센서 등 연결    |
| USB HID        | USB 키보드/마우스 시뮬레이션              |
| SPI/I2C 통신   | 외부 칩 및 디바이스와 통신                |
| PIO 프로그래밍 | 고속 비트 제어, 맞춤형 프로토콜 구현 가능 |

---

## 📚 6. RP2040의 특징 (차별점)

- **듀얼 코어**: 멀티태스크 가능
- **PIO 블록**: SPI, I2C, PWM 등 직접 구현 가능
- **정밀 타이머와 클럭 제어**
- **저전력 설계**: 배터리 기반 시스템에 적합

---

## 🔍 7. Raspberry Pi Pico vs Arduino

| 항목      | Raspberry Pi Pico | Arduino Uno (ATmega328P) |
| --------- | ----------------- | ------------------------ |
| 코어 수   | 2개 (M0+)         | 1개                      |
| 클럭 속도 | 최대 133 MHz      | 16 MHz                   |
| SRAM      | 264 KB            | 2 KB                     |
| 가격      | 약 \$4            | 약 \$20                  |
| USB 기능  | 디바이스/호스트   | 디바이스 전용            |
| 언어      | C++, MicroPython  | Arduino C                |

---

## 🎯 8. 추천 사용

- **초소형 MCU 프로젝트**를 저렴하게 시작하고 싶은 개발자
- **MicroPython**으로 빠르게 실습하고 싶은 개발자
- 고속 GPIO 제어(PIO)가 필요한 고급 사용자
- 기존 아두이노로 한계 느낀 사용자 → **Pico로 확장**

## ✅ 결론

> **라즈베리파이 피코는**
> 아두이노처럼 MCU를 제어하지만, **훨씬 더 강력하고 저렴하며**
> C/C++ 뿐 아니라 MicroPython으로도 쉽게 접근할 수 있는 범용 임베디드 플랫폼
